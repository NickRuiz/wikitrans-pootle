About Pootle

Pootle is a simple web portal that should allow you to translate! Since Pootle is Free Software, you can download it and run your own copy if you like. You can also help participate in the development in many ways (you don't have to be able to program).
The Pootle project itself is hosted at translate.sourceforge.net where you can find the details about source code, mailing lists etc.
The name stands for PO-based Online Translation / Localization Engine, but you may need to read this.
